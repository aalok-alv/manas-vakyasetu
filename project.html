What is Vakyasetu?

Vakyasetu is a modular IoT-based assistive communication system designed to empower individuals with paralysis or severe speech impairment to communicate independently. The system converts minimal physical movements detected through adaptable sensors into text and speech using an intelligent alphabet-grid interface, NLP-based word prediction, and text-to-speech synthesis. Built on a wireless IoT architecture using NodeMCU, the platform allows flexible integration of different input sensors based on the patient's physical capabilities, making it highly customizable and patient-centric. By combining low-cost hardware with intelligent software, Vakyasetu provides an affordable, scalable, and accessible communication solution for healthcare, rehabilitation, and home-care environments.
